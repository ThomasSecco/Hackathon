# 🚀 Optimizing Supply Chains with AI 🌎

Welcome to **Diversions.AI**! This project was created during the **GreenHack~IT 2024** hackathon, hosted by **Ariane Group**. Our solution demonstrates the power of **Machine Learning** and **Data Analytics** in optimizing supply chains for improved **sustainability** and a better future. By leveraging AI, we aim to reduce CO₂ emissions and enhance resource utilization.

## 📜 Project Overview
This repository contains the resources and code used to:
1. **Predict transportation and lead times** with an ML model.
2. **Optimize CO₂ emissions** while meeting delivery constraints.
3. **Provide insights** for better decision-making across the supply chain.

## 🧩 Key Components
### ⚙️ Machine Learning Pipeline
- **Sanity Checks** and **Data Preprocessing** for quality data input
- **Feature Engineering**: Added 20+ features, including environmental and operational metrics
- **Model Choice**: XGBoost regressor to predict lead times accurately

### 📊 Optimization
- **Objective**: Minimize CO₂ emissions and optimize delivery
- **Constraints**: Incorporates ship availability, container numbers, and travel time
- **Solution**: Uses MILP (Mixed Integer Linear Programming) for optimal scheduling

### 🌐 Frontend & Backend
- Interactive web application for **visualizations** and **user input handling**
- Integration with external data sources for real-time analysis

## 📅 Future Enhancements
- **Supply Chain Resilience**: Scenario planning and sustainability forecasting
- **Lifecycle Analysis** for resource efficiency
- **Supply Chain Transparency**: Tracking environmental impacts and performance metrics
